## Backend

The backend runs on **Cloudflare Workers** with **D1** (SQLite-compatible) for storage.

Responsibilities:
- Invite code issuance and validation (Mainnet closed beta)
- Token price caching
- API layer between frontend and on-chain state

Key design notes:
- D1 storage limit: 10GB hard cap. The `token_prices` table is separated into its own DB to manage scale.
- Invite codes are admin-only issued, wallet-bound, single-use, stored on-chain and in D1.

## Environment Separation

Devnet and Mainnet run as completely separate deployments:

| URL | Network | Access |
|---|---|---|
| `axs.pizza` | Solana Devnet | Open |
| `mainnet.axs.pizza` | Solana Mainnet | Invite code required |

The backend at each environment points to the corresponding program IDs and oracle accounts.
