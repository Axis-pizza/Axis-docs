Mainnet access is invite-only during the closed beta. You need:

1. An **invite code** — issued by the Axis team

## Redeeming Your Invite Code

1. Go to [mainnet.axs.pizza](https://mainnet.axs.pizza)
2. Connect your wallet
3. Enter your invite code when prompted
4. Your wallet address is now bound to the code — access is permanent

Each invite code is single-use and wallet-bound. It cannot be transferred after redemption.
